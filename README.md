## ğŸš€ Getting Started

### Prerequisites

- Node.js (v16 or higher)
- npm or yarn
- Git

### Installation

1. **Clone the repository**
   ```bash
   git clone https://github.com/adiyaan/portfolio.git
   cd portfolio
   ```
2. **Install dependencies**
   ```bash
   npm install
   # or
   yarn install
   ```
3. **Run the development server**
   ```bash
   npm run dev
   # or
   yarn dev
   ```
4. **Open your browser and navigate to `http://localhost:3000`**
